# SEO Analysis Dashboard

A dashboard to show analysis and visualisation for CSV files as extracted from google cursor files. Users download csv file of SEO statistics from Google Cursor, upload it to their file and then get all statistical information for that available website.

# Tools and Technologies:

## Frontend:
- ReactJS
- NextJS
- TailwindCSS
- Javascript

## Backend:
- Python
- FastAPI

## Data Analysis:
- Python Jupyter Notebook
- Numpy
- Pandas
- Matplotlib
